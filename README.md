# My Dotfiles

The following repository contains my dotfiles that I use on Linux. It only contains the files for programs that are kept identical across installations (e.g. Neovim, Emacs), and not for programs that I might keep different setups on depending on the device. As an example I don't currently keep any compositors' configs (e.g. Hyprland, Niri etc.) in this repo as I have different setups for my laptop and desktop (although this may change at some point in the future).

## Installation

Simply copy the files to the `~/.config` folder, then run the `install.sh` script to install any necessary dependencies. Note that `yay` is required for installing dependencies.
